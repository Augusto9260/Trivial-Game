# 🌟 Trivial Game

<p align="center">
  Primeira versão estável do Trivial Game, um jogo de perguntas e respostas dinâmico e competitivo construído para dispositivos Android.
</p>

---

## 🚀 Funcionalidades
* **Perguntas em Tempo Real:** Integração completa com API de perguntas.
* **Interface Moderna:** Tema escuro (Dark Mode) com paleta ciano e azul de alta fidelidade.
* **Identidade Visual Única:** Logotipo exclusivo em formato de estrela medalha desenvolvido no Figma.

## 📱 Como Jogar (Download do APK)
Você não precisa compilar o código para testar! A versão de produção já está pronta e empacotada.
1. Acesse a aba [Releases](https://github.com/Augusto9260/Trivial-Game/releases) deste repositório.
2. Baixe o arquivo `TrivialGamev1.0.0.apk`.
3. Transfira para o seu telemóvel Android e faça a instalação.

## 🛠️ Tecnologias Utilizadas
* **Framework:** React Native
* **Linguagem:** JavaScript
* **Gerenciador de Dependências:** NPM / Yarn
* **Automação de Build:** Gradle (Android)
* **Design/UI:** Figma

## 💻 Como rodar o ambiente de desenvolvimento
Se quiser clonar o repositório e rodar o projeto localmente:

```bash
# Clonar o repositório
git clone [https://github.com/Augusto9260/Trivial-Game.git](https://github.com/Augusto9260/Trivial-Game.git)

# Entrar na pasta
cd Trivial-Game

# Instalar as dependências
npm install

# Iniciar o Metro Bundler
npm start

# Executar no Android (com o emulador aberto)
npm run android
